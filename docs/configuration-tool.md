# Configuration Tool

<a href="../vflex-tool.html">Launch the Configuration Tool</a>

The Configuration Tool is a browser-based interface for reading and writing VFLEX device parameters. It allows you to:

- View device information (serial number, hardware ID, firmware version)
- Set output voltage and current limit
- View voltage limits
- Read the USB PD Power Data Object (PDO) log
- Clear the PDO log and enable scan mode
